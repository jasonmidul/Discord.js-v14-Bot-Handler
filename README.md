## Features

- Developed using Discord.js v14
- Multi-language feture with database
- Coustom client
- Event handler
- Button handler
- Modal handler
- Autocomplete handler
- Advanced slash command handler
- Events and commands reload command
- hybrid-sharding
- Dev guild only command
- Guild premium system
- User premium system
- Eval slash command
